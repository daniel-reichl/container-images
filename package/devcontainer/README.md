# Devcontainer Image

Published development image for VS Code Dev Containers, Codespaces, and other devcontainer-compatible tooling.

`ghcr.io/daniel-reichl/devcontainer`

## Use It

Reference the image directly in `.devcontainer/devcontainer.json`:

```jsonc
{
  "image": "ghcr.io/daniel-reichl/devcontainer:latest"
}
```
